Upload files to Amazon S3

This is an implementation of a simple GUI for uplodng files to Amazon S3. All work is done client-side, using Amazon as an identity provider, and the AWS JavaScript SDK.

This page uses only client-side code to authenticate and upload files to S3. We have not written any server-side code at all. In fact, the entire example is hosted in an S3 bucket. Try it out by clicking on Amazon identity providers to the right.
